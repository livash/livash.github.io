Welcome to the Asteroids game

How to play:

  • Use up, down, left and right key to move the ship;
  • Use space bar to fire bullets;


  • "Score" keeps track of the number of destroyed asteroids.
  • Game is over once the ship collides with an asteroid.
  • Press "Start" to start the Game.

Technical Features:

  • Game written in JavaScript GitHub
  • HTML5 canvas
