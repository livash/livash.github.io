    Flickr-Light GitHub

    The latest web application I made is called Flicker-Light. It mimics key functionalities of Yahoo's Flickr.com and it is hosted on heroku flickr-light.herokuapp.com. My Flicker-Light app can support multiple users, who can upload and view uploaded photos, tag photos, leave comments on the photo, and change photo's description, as well as organize photos into albums and galleries to shear with other users. I use Amazon's S3 cloud storage to store user's media files. In contrast to the original Flickr, my Flicker light has some interesting CSS3 animations and styling. I also made a heavy use of JavaScript and jQuery.ui to make the app mode interactive and dynamic. Below are some screenshots for this project. DO checkit out on heroku. Use GUEST account.

    Key Technical Features: Rails 3.2, Ruby, JavaScript, jQuery, jQuery.ui, Devise, Paperclip, AWS, CSS3, Animate.css, Bootstrap.
